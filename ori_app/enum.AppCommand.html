pub fn open_window<V: View + 'static>( window: Window, view: impl FnMut() -> V + Send + 'static, ) -> Self

Convenience method to open a window with a view.

Note that V must implement View<()>, and therefore cannot access the data of the application. If you need to access the data, you can implement a Delegate instead.

§Example
fn ui() -> impl View {
    // Here we create a button that opens a new window when clicked.
    on_click(
        button(text("Open new window")),
        |cx, _| {
            let window = Window::new()
                .title("New window");

            cx.cmd(AppCommand::open_window(window, popup));
        },
    )
}

fn popup() -> impl View {
    text("Hello, world!")
}
