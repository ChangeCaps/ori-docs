Struct ori_font_awesome::Icon

pub struct Icon {
    pub icon: IconCode,
    pub solid: bool,
    pub size: Styled<f32>,
    pub color: Styled<Color>,
}
Expand description

A view that displays a single icon.

By default, the icon is rendered using the icon.font font family. This uses the Font Awesome 6 Regular Free font by default.

Fields§

§icon: IconCode

The codepoint of the icon to display.

§solid: bool

Whether the icon is solid or regular.

This only affects the rendering of the icon if the icon is available in both styles.

§size: Styled<f32>

The size of the icon.

§color: Styled<Color>

The color of the icon.
